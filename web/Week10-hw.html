Homework2: brief introduction

  Play framework is used here, and a simple survey is achieved. I implement 'add' and 'show' operations, and 'delete' is not achieved due to time-scheduling. Important work is app/controllers/Application.java, app/models/Person.java and views/index.scala.html, while there are also subtle changes to conf/application.conf and conf/routes.   My web application executes in port 9119, and I reconfig the apache to suport reverse-proxy.(dynamicly add proxy-module, the virtual host is executed in port 8619.) As a result, the command to start the application will be: ./play-java -Dhttp.port=9119 -DapplyEvolutions.default=true   To use the application, type in url 162.105.146.180:8619
