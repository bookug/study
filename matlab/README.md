折线图

```
 x=[1,5,3,7,4]

  plot(x)

  在上面的例子中自变量默认是从1到5

  plot(x, '-*')

  x=[1,2,3,4,5]

   y=[1,2,3,4,5]

    z=[6,5,4,3,2]

    plot(x,y,'-s',x,z,'-*')

```

柱状图

x = [2010;2011;2012;2013;2014;2015;2016;2017];

y1 = [100;67;90;88;89;95;80;82];

bar(x,y1)

如果需要绘制将两个或者多个柱状图组成一个大的柱状图，按照下面操作。

输入:

x = [2010;2011;2012;2013;2014;2015;2016;2017];

y1 = [100;67;90;88;89;95;80;82];

y2 = [123;110;150;119;134;129;109;142];

y = [y1,y2];

bar(x,y)


可以使用xlabel，ylabel为X轴和Ｙ轴添加注释

例如：

x = [2010;2011;2012;2013;2014;2015;2016;2017];

y1 = [100;67;90;88;89;95;80;82];

bar(x,y1)

xlabel('年份')

ylabel('学生成绩')


title('学生历年成绩')

可以使用legend为柱状图添加图注

legend('化学成绩')
